USB-MIDI Hack kit for Arduino Uno R3

May 17, 2015

With USB - MIDI Hack kit for Arduino Uno R3, you can immediately create a device using USB - MIDI.

Although you can rewrite MocoLUFA's firmware yourself, since it seems that thresholds are high for beginners, we decided to distribute development kits along with the switch.

1. Contents of kit

USB-MIDI Hack kit for Arduino Uno R3 includes the following items.

  • Arduno Uno R3 (DIP version) Updated firmware to MocoLUFA
  • MocoLUFA firmware · selector switch board

2. How to use

  1. Insert the selector switch board into the ISP connector near the USB connector.
  2. Switching the slide switch and pressing the reset button switches the mode.
  3. Develop sketches while switching modes.
  4. After development, if you remove the selector switch board, it becomes a USB-MIDI device.
  • USB-MIDI mode

If it is set to USB side, it will be in USB-MIDI mode. (The LED on the selector switch board lights up.)

In this mode, the PC recognizes it as a USB-MIDI device. You can send data from DTM software etc.

With Arduino sketches, you can create programs using MIDI libraries.

  • USB-Serial mode

If it is on the opposite side of USB, it will be in USB - Serial mode. (The LED on the selector switch board goes out.)

In this mode you can write sketches from the Arduino IDE.

3. TIPS / bonus

3.1 High speed mode

When shorting the jumper of HI - SP, it communicates with Arduino 's sketch at a speed of 1 Mbps. To use this mode, it is necessary to change the sketch and specify 1 Mbps speed.

When sending a large amount of data, there is a possibility that data loss may be reduced.

3.2 Combined use with shield

Although the shape of the switch is designed so as not to disturb the shield, there is a possibility of interference in the height direction.

In case of interfering, squatting up sockets only during development will not get in the way.

3.3 When to DIY

The source code of MocoLUFA is available from GitHub .
